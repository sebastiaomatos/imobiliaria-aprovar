# Logos para Google Performance Max (assets de logo)

O Google pede o logo como asset separado, em dois formatos:
- `aprovar-pmax-logo-1x1.png` — **1200×1200 (1:1)** — quadrado (slot obrigatório de logo).
- `aprovar-pmax-logo-4x1.png` — **1200×300 (4:1)** — paisagem (slot recomendado).

Fundo areia (#F7F5EE) para leitura consistente nas várias colocações do Google. Se preferir
que o Google adapte ao fundo, há versões transparentes em `_templates/branding/`.

## Onde usar
Google Ads → campanha Performance Max → grupo de recursos → seção **Logotipos** → enviar os dois.

## Onde salvar no repositório
_templates/branding/pmax/
