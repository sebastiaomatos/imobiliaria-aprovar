# 01 · Apostila do método

`Apostila-Anuncios-que-Vendem-Imoveis.pdf` (29 páginas) — ensina, do zero, o método de venda de
imóveis por Marketing de Intenção: mentalidade, algoritmo, os 3 elementos (ICP, imóvel protagonista,
qualificativos), anatomia do anúncio + 30 ganchos, 3 formatos, Meta Ads do zero, script de WhatsApp,
escala e checklist.

Fontes em `src/`: `apostila.html` + `make_diagrams.py` (10 infográficos) + `img/`. Regenerar o PDF:
`weasyprint src/apostila.html Apostila-Anuncios-que-Vendem-Imoveis.pdf`.
