# 09 · Playbook acionável — AI Listings (captação com IA)

Reconstrução **executável** do funil por trás do AI Listings™ (re-leads.store), sem o curiosity gap.

**Entregável:** [`Playbook-AI-Listings-Captacao.pdf`](Playbook-AI-Listings-Captacao.pdf) (12 págs).

## O funil
`Anúncio (proprietários) → Landing de avaliação grátis → Laudo/CMA por IA → follow-up + qualificação → reunião → captação assinada`, com **ChatGPT** produzindo criativos, laudo, follow-ups, scripts de objeção e a apresentação de captação.

## Conteúdo da pasta
| Caminho | O que é |
|---|---|
| `Playbook-AI-Listings-Captacao.pdf` | Playbook ilustrado (funil, IA, anúncios, landing, prompts, automação, métricas) |
| `prompts/chatgpt-prompts.md` | **Biblioteca de 6 prompts** prontos (anúncio, CMA, follow-up, objeções, listing presentation, qualificação) |
| `anuncios/ad-copy.md` | Headlines/copy/CTAs (EN + PT-BR) + público + compliance |
| `automacao/seller-followup.md` | Régua de follow-up do vendedor (0min→Dia 14) com mensagens |
| `img/` | 4 diagramas + 3 criativos de exemplo |
| `src/` | Geradores (`make_assets.py`, `playbook_ai.html`) |

## Notas
- Prompts/copy em **inglês** (mercado-alvo do produto é global); cada arquivo traz **adaptação PT-BR** (ex.: SMS → WhatsApp, "home valuation" → "avaliação do seu imóvel").
- **Realidade vs. marketing:** o método exige **verba de mídia** (anúncios) — os exageros "sem budget / 90% / no automático" do produto original foram descartados.
